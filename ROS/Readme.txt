Ветка посвящена разработки под ROS для проекта по созданию единого графеческого интерфейса для различных алгоритмов роевго управления БЛА. 

В силу того, что до публикации материал закрытый, могу представить в открытом доступе только базовую ноду которая была написана в процессе знакомства с ROS
и обучению работе с пакетом PX4-LIRS. Нода представляет из себя код, который заставляет дрон взлететь, и перелететь по заданны координатам.
